# Wainaina Thomas — Product Designer Portfolio

This is the source code for my professional product design portfolio, featuring case studies on mobile-first solutions and functional web applications.

## Projects

- **Dill**: A supermarket deal aggregator for the Kenyan market.
- **Nairobi National Museum**: A redesigned ticketing flow focused on guest checkout and international accessibility.

## Tech Stack

- **Framework**: [Next.js 16](https://nextjs.org/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Animation**: [Framer Motion](https://www.framer.com/motion/)
- **Deployment**: [Vercel](https://vercel.com/)

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

This project is configured for one-click deployment on Vercel. Connect the GitHub repository to your Vercel account to enable automatic deployments on every push to `main`.
